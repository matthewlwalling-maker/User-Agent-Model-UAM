# Sentinel Baseline Packet v0.1

This bundle contains a pre-compression test execution packet and downstream prompt.

Files:
- `SENTINEL_BASELINE_EXECUTION_PACKET_v0.1.md` — full run contract and cases.
- `SENTINEL_BASELINE_DOWNSTREAM_PROMPT_v0.1.md` — compact prompt for the downstream ChatGPT agent.

Use before any source compression/merge exercise. The run creates a behavioral fingerprint for high-risk rules so the compressed package can be regression-tested against the same cases.
